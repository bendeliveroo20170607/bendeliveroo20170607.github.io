Text Shadow

You can give text an interesting special effect using the text-shadow property:

h1 {
  text-shadow: 5px 5px 10px green;
}

This would then look like this:

The syntax of this rule may need a little explanation, as it contains four important parts, each separated by a space:

h1 {
  text-shadow: 5px 5px 10px green;
}

The first number represents the offset of the text shadow relative to the text it appears behind, in the x-axis (i.e. horizontally), in a distance unit (such as a number of pixels).

h1 {
  text-shadow: 5px 5px 10px green;
}

The second number represents the offset of the text shadow in the y-axis (i.e. vertically).

h1 {
  text-shadow: 5px 5px 10px green;
}

The third number represents the radius of the blur effect on the text shadow. You can set this to 0 or omit it entirely to have a shadow that appears as sharp as the text itself.

h1 {
  text-shadow: 5px 5px 10px green;
}

The final part of the rule is the colour you wish the shadow to appear in, which can be any named colour or explicit value (in hexadecimal, rgb(), rgba() or hsl() form).
