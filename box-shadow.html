The syntax of this rule may need a little explanation, as it contains four important parts, each separated by a space:

img {
  box-shadow: 5px 5px 10px grey;
}

The first number represents the offset of the box shadow relative to the element it appears behind, in the x-axis (i.e. horizontally), in a distance unit (such as a number of pixels).

img {
  box-shadow: 5px 5px 10px grey;
}

The second number represents the offset of the box shadow in the y-axis (i.e. vertically).

img {
  box-shadow: 5px 5px 10px grey;
}

The third number represents the radius of the blur effect on the box shadow. You can set this to 0 or omit it entirely to have a shadow that appears solid.

img {
  box-shadow: 5px 5px 10px grey;
}

The final part of the rule is the colour you wish the shadow to appear in, which can be any named colour or explicit value (in hexadecimal, rgb(), rgba() or hsl() form).
