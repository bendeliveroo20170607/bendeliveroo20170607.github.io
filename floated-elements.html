Floated Elements

By default, most HTML elements have one of two layout modes: inline, or block. Elements that have inline layout will appear on the same horizonal row as other elements that have this layout mode; for example, the em and strong tags used to emphasise text. Conversely, block display elements appear below one another vertically, for example a p tag for a new paragraph. But what if you want a block display element to appear alongside another element instead of below it?

There are lots of ways of achieving this, and the best way depends a lot on what kind of things you’re laying out. For a really simple example, let’s take our portrait photo and about me paragraph:

<img src="ben.jpg" height="200" width="200">

<p>
  Hi! My name is Ben. I’m a front-end engineer at Deliveroo,
  working on the Growth team. Our team is responsible for
  trying out all sorts of experiments to see what can make
  Deliveroo a better experience for our customers!
</p>

An img element will appear inline by default, but the paragraph tag below it will have block layout, forcing it to appear underneath the photo, which would make it look like this:

A simple way of changing how these two elements layout respective to each other is to give the image a float layout property, like this:

img {
  float: left;
}

Once we do this, the image ‘floats’ to the left of its container, and the paragraph of text slides up alongside it:

As you can see, this still doesn’t look very nice — the text is aligned flush to the right hand edge of the portrait photo, but we can fix this by adding a margin on the edge of the image:

img {
  float: left;
  margin-right: 30px;
}

Voila!
