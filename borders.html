Borders

You can give any element an outline border using the border property:

img {
  border: 5px solid grey;
}

The three components of this rule are (in order): width of the border, style of border (solid, dashed, dotted etc.) and then finally the colour of border which can be any named colour or explicit value (in hexadecimal, rgb(), rgba() or hsl() form).

You can also give elements a border-radius, to give them rounded corners. Despite the name including the word ‘border’, elements with a border-radius property don’t actually need to have a border as well:

img {
  border-radius: 20px;
}
