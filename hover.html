Hover

The appearance of elements can also be made to change when you move your mouse cursor over them, using the CSS :hover ‘pseudo-class’. The way you use this is to combine it with the selector you want to target, by writing it after the selector name, like this:

h1 {
  font-size: 40px;
}

h1:hover {
  color: red;
}
